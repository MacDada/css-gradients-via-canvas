CSS Gradients via Canvas

This project seeks to implement WebKit's CSS Gradients proposal in browsers that do implement the canvas API. Simply include css-gradients-via-canvas.js on a page that has document.querySelectorAll available (looks for jQuery, Prototype, or Sizzle otherwise to query selectors).

Works in Firefox 3.5, but should also work in Firefox 3.0 and Opera.
